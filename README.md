# Consequence Engine Demo

A sample project for testing Permission Protocol's consequence analysis on pull requests.

## Structure

- `src/lib/auth/` — Authentication and session management
- `src/lib/billing/` — Stripe payment integration

## Getting Started

1. Install the [Permission Protocol GitHub App](https://github.com/apps/permission-protocol)
2. Open a pull request
3. See the consequence analysis appear automatically
